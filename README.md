# Vovel
voice based chatfiction

> 프로젝트 간략 소개 한 문장 
- 
- 오디오 기반 채팅형 소설 프로젝트입니다. 

## 핵심 기능  Key Feature
- 줄글로 된 고전 소설을 채팅형식으로 변환합니다. 
- 채팅형식으로 변환된 소설을 오디오로 만들 수 있습니다.
